dot product: takes two vectors and returns a scalar:
if result is > 0, they point the same way.
if result is = 0, they are perpendicular at 90 degrees.
if result is < 0, they point away from each other.
this is used to calculate how much light hits a surface, if the light vector anc surface vector point in
opposite directions the surface is lit up.
it can also check if a ray is hitting the front or back of an object.

Cross product: takes two vectors and returns another vector:
the new vector is perpendicular (90 degrees) to both of the vectors, used to calculate surface normals,
which is the way a triangle is facing in a 3d space, allowing to show which way a surface is facing.
